Hướng dẫn chạy chương trình demo

1. Cần chạy trên web server vì chạy trên file dạng local thì webrtc sẽ không hoạt động
2. Thư mục Demo/PT : hoạt động dựa trên pubnub, cần có internet
3. Thư mục Demo/PTLocal : hoạt động dựa trên websocket, sử dụng NodeJS với đoạn mã reflector.js như sau
  node.exe reflector.js
  
4. File reflector.js nằm trong thư mục Demo/PTLocal
